## 개발 결과물을 사용하는 방법 소개

### 설치 방법

1. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

2. 환경 변수 설정:
- `.env` 파일을 생성하고 다음 내용을 추가하세요:
```
OPENWEATHER_API_KEY=your_api_key_here
```
- `your_api_key_here`를 OpenWeather API 키로 교체하세요.

### 실행 방법

```bash
python -m flask run
```
